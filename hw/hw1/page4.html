Go doesn't include inheritance and instead provides 2 alternative methods. The first is embedding also known as composition, which is essentially combining two classes into one. And the second is interfaces, which is a class which has 2 types. It's an interface type as well as another specified class type, allowing for functions and variables to be shared. Additionally, Go does not allow generic programming, which are function or object types that are specified later on in the program.
