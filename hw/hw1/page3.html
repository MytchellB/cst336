Go achieves concurrency with goroutines and channels, which allows for CPU parallelism and running database operations while the rest of the prorgam runs simultaneously.
Goroutines are called with go keyword, which then allows you to write a function using multiple threads.
Channels can be denoted as chan T which specifies a channel that can only transmit messages of type T. All of this allows you to utilize multiple threads on a CPU, and run multiple operations concurrently quite easily. Especially when compared to the languages Go is attempting to improve upon.
